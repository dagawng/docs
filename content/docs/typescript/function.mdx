### 1.Basic Function Declaration

```typescript
// Array of fruit objects
const fruits = [
  { name: "apple", color: "red", quantity: 10 },
  { name: "banana", color: "yellow", quantity: 5 },
  { name: "orange", color: "orange", quantity: 8 },
];

// Function to check if a fruit is available in the list
function isFruitAvailable(fruitName: string): boolean {
  return fruits.some((fruit) => fruit.name === fruitName);
}

// Function to get fruit details
function getFruitDetails(fruitName: string): void {
  const fruit = fruits.find((fruit) => fruit.name === fruitName);
  if (fruit) {
    console.log(`✔️ ${fruit.name} is available.`);
    console.log(`   ➤ Color: ${fruit.color}`);
    console.log(`   ➤ Quantity: ${fruit.quantity}`);
  } else {
    console.log(`❌ ${fruitName} is not in stock.`);
  }
}

// Try checking for a fruit
const fruitToCheck = "banana";

if (isFruitAvailable(fruitToCheck)) {
  getFruitDetails(fruitToCheck);
} else {
  console.log(`${fruitToCheck} is not available in the fruit list.`);
}
```

### 2.Function Type Expressions

function တစ်ခုရဲ့ parameter တွေ၊ return type တွေကို သီးသန့်ဖော်ပြဖို့ အသုံးပြုတဲ့ type expression တစ်ခုပဲဖြစ်ပါတယ်။

```typescript
// Define a function type for mathematical operations
let add: (a: number, b: number) => number;
let subtract: (a: number, b: number) => number;
let multiply: (a: number, b: number) => number;
let divide: (a: number, b: number) => number;

// Assign actual functions
add = function (x, y) {
  return x + y;
};

subtract = (x, y) => x - y;

multiply = (x, y) => x * y;

divide = (x, y) => {
  if (y === 0) {
    console.log("Cannot divide by zero!");
    return 0;
  }
  return x / y;
};

// Function that takes another function as parameter
function calculate(
  a: number,
  b: number,
  operation: (x: number, y: number) => number
): number {
  return operation(a, b);
}

// Test the functions
console.log("Add: ", calculate(10, 5, add)); // 15
console.log("Subtract: ", calculate(10, 5, subtract)); // 5
console.log("Multiply: ", calculate(10, 5, multiply)); // 50
console.log("Divide: ", calculate(10, 0, divide)); // Error msg + 0
```

### 3.Optional Parameters `?`

TypeScript မှာ parameter တစ်ခုကို optional ဖြစ်အောင် သတ်မှတ်ချင်လျှင် parameter name နောက်မှာ <span style={{ color: 'red' }}>?</span> သင်္ကေတကို ထည့်ရပါတယ်။
ဒါကအဓိပ္ပါယ်မှာ function ကိုခေါ်တဲ့အချိန်မှာ အဲဒီ argument ကို မပေးလည်းရတယ် ဆိုတာကိုဖော်ပြတာပါ။

```typescript
// 1. Simple example with one optional parameter
function greetUser(name?: string): void {
  if (name) {
    console.log(`Hello, ${name}!`);
  } else {
    console.log("Hello, guest!");
  }
}

greetUser("John"); // Hello, John!
greetUser(); // Hello, guest!

// 2. Optional middle parameter
function createUser(username: string, age?: number): string {
  if (age !== undefined) {
    return `User: ${username}, Age: ${age}`;
  } else {
    return `User: ${username}, Age: not provided`;
  }
}

console.log(createUser("Alice", 25)); // User: Alice, Age: 25
console.log(createUser("Bob")); // User: Bob, Age: not provided

// 3. Optional parameter with default value
function sendMessage(message: string, sender: string = "System"): void {
  console.log(`[${sender}] ${message}`);
}

sendMessage("Welcome!"); // [System] Welcome!
sendMessage("Hi", "Admin"); // [Admin] Hi

// 4. Optional parameter with complex logic
function calculatePrice(basePrice: number, discount?: number): number {
  if (discount !== undefined) {
    return basePrice - (basePrice * discount) / 100;
  }
  return basePrice;
}

console.log(calculatePrice(100)); // 100
console.log(calculatePrice(100, 10)); // 90
```

### 4.Default Parameters

Default parameter ဟု ဆိုသည်မှာ function ကိုခေါ်သည့်အခါ argument မပါဝင်လည်း၊ parameter အတွက် သတ်မှတ်ထားသော မူလတန်ဖိုးကို အလိုအလျှောက်သုံးပေးခြင်း ဖြစ်သည်။

```typescript
// 1. Default value for a simple greeting
function greetUser(name: string = "Guest"): void {
  console.log(`Hello, ${name}!`);
}

greetUser(); // Hello, Guest!
greetUser("John"); // Hello, John!

// 2. Default numeric value
function calculateDiscount(price: number, discount: number = 10): number {
  return price - (price * discount) / 100;
}

console.log(calculateDiscount(100)); // 90
console.log(calculateDiscount(100, 20)); // 80

// 3. Default boolean value
function toggleSetting(settingName: string, isEnabled: boolean = true): void {
  console.log(`${settingName} is now ${isEnabled ? "enabled" : "disabled"}.`);
}

toggleSetting("Dark Mode"); // Dark Mode is now enabled.
toggleSetting("Notifications", false); // Notifications is now disabled.

// 4. Multiple default values
function createUser(
  username: string = "anonymous",
  age: number = 0,
  role: string = "user"
): void {
  console.log(`Username: ${username}, Age: ${age}, Role: ${role}`);
}

createUser(); // Username: anonymous, Age: 0, Role: user
createUser("Alice"); // Username: Alice, Age: 0, Role: user
createUser("Bob", 30, "admin"); // Username: Bob, Age: 30, Role: admin
```

### 6.Void Return Type

void type ဆိုတာ function က တစ်ခုခုလုပ်ပေမယ့် return တန်ဖိုးမရှိတဲ့အခါ သတ်မှတ်ရာမှာ သုံးပါတယ်။

```typescript
function logMessage(message: string): void {
  console.log(message);
}
```

### Rest Parameters

Rest parameters က argument အများကြီးကို array တစ်ခုထဲ စုပေါင်းပေးနိုင်စေပါတယ်။

```typescript
function sum(...numbers: number[]): number {
  return numbers.reduce((total, n) => total + n, 0);
}

console.log(sum(1, 2, 3)); // ➜ 6
console.log(sum(10, 20)); // ➜ 30
```

### 8.Function Overload

multiple function signature for different call types

```ts
function combine(a: string, b: string): string;
function combine(a: number, b: number): number;
function combine(a: any, b: any): any {
  return a + b;
}

combine("Hello, ", "World!"); // "Hello, World!"
combine(5, 10); // 15
```

### 9.Arrow Function

Same typing applies to arrow functions:

```typescript
const multiply = (x: number, y: number): number => x * y;
```

### 10.Object as Parameters and excess property checking

```ts
// Function that accepts a student object with only id, name, and age
function createStudent(student: { id: number; name: string; age: number }) {
  console.log(`${student.name.toUpperCase()} is ${student.age} years old.`);
  // Output will depend on the 'student' object passed
}

// ✅ Example 1: Correct object — only expected properties
const student1 = { id: 1, name: "Alice", age: 20 };
createStudent(student1);
// Console Output: ALICE is 20 years old

// ❌ Example 2: Direct object literal with an extra 'email' property
createStudent({ id: 2, name: "Bob", age: 22, email: "bob@example.com" });
// Error:
// Argument of type '{ id: number; name: string; age: number; email: string; }' is not assignable to parameter of type '{ id: number; name: string; age: number; }'.
// Object literal may only specify known properties, and 'email' does not exist in type ...

// ✅ Example 3: Assign to a variable first — excess property allowed
const student3 = { id: 3, name: "Carol", age: 21, email: "carol@example.com" };
createStudent(student3);
// Console Output: CAROL is 21 years old

// ✅ Example 4: Use type assertion to ignore excess property (use with caution)
createStudent({ id: 4, name: "Dave", age: 19, email: "dave@example.com" } as {
  id: number;
  name: string;
  age: number;
});
// Console Output: DAVE is 19 years old
// Console Output:
// ALICE is 20 years old.
```

### Type Alias and Intersection Type

```ts
// ✅ Type Alias: တစ်ခုသော type ဖွဲ့စည်းပုံအတွက် အမည်တစ်ခုကို သတ်မှတ်ပေးခြင်း ဖြစ်သည်။

type Person = {
  name: string;
  age: number;
};

// ✅ Another Type Alias for Contact Information
type Contact = {
  email: string;
  phone: string;
};

// ✅ Intersection Type: combines multiple types into one
// ဤသည်မှာ Person နှင့် Contact မှ property များအားလုံးပါဝင်ရမည့် type အသစ် တစ်ခုကို ဖန်တီးခြင်းဖြစ်သည်။
type Employee = Person & Contact;

function printEmployeeInfo(emp: Employee) {
  console.log(`Name: ${emp.name}`);
  console.log(`Age: ${emp.age}`);
  console.log(`Email: ${emp.email}`);
  console.log(`Phone: ${emp.phone}`);
}

// ✅ Example object that matches the Employee type (must include all fields from Person & Contact)
const employee1: Employee = {
  name: "Alice",
  age: 28,
  email: "alice@example.com",
  phone: "123-456-7890",
};

printEmployeeInfo(employee1);

/*
Expected Console Output:
Name: Alice
Age: 28
Email: alice@example.com
Phone: 123-456-7890
*/
```

### 📝 အနှစ်ချုပ် (Function + Type Alias + Intersection Type)

| အကြောင်းအရာ                  | ရှင်းလင်းချက်                                                       | ဥပမာ                                                                |
| ---------------------------- | ------------------------------------------------------------------- | ------------------------------------------------------------------- |
| **Function Declaration**     | Function ကို တည်ဆောက်သတ်မှတ်ခြင်း                                   | `function greet(name: string): void { console.log("Hi " + name); }` |
| **Function Type Expression** | Function ရဲ့ type ကို သီးခြားသတ်မှတ်နိုင်သည်                        | `type GreetFunc = (name: string) => void;`                          |
| **Optional Parameter**       | `?` ဖြင့် optional parameter သတ်မှတ်နိုင်သည်                        | `function greet(name?: string) {}`                                  |
| **Default Parameter**        | မပေးလျှင် default value သတ်မှတ်ထားနိုင်သည်                          | `function greet(name = "Guest") {}`                                 |
| **Rest Parameter**           | `...` အသုံးပြု၍ parameter များစွာကို array တစ်ခုအဖြစ် သိမ်းနိုင်သည် | `function sum(...nums: number[]) {}`                                |
| **Void Return Type**         | မည်သည့်တန်ဖိုးကိုမျှ return မပြုသော function                        | `function log(msg: string): void {}`                                |
| **Type Alias**               | Data structure တစ်ခုကို နာမည်တစ်ခုဖြင့် သတ်မှတ်ခြင်း                | `type User = { name: string; age: number };`                        |
| **Intersection Type (`&`)**  | Type နှစ်ခု (သို့ များ) ကို ပေါင်းပြီး အသစ်တစ်ခုဖန်တီးခြင်း         | `type Contact = Person & Address;`                                  |
