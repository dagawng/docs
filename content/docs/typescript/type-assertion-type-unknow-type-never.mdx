### Type Assertion (အမျိုးအစားသတ်မှတ်ခြင်း)

TypeScript ကို ဤ value ရဲ့အမျိုးအစားကို ကိုယ်တိသိပြီးသားဖြစ်တဲ့အခါ အသုံးပြုသည်။

#### 📌 ဥပမာ (၁) - any ကို string အဖြစ်သတ်မှတ်

```ts
let someValue: any = "Hello TypeScript";

// 'as' ဖြင့်သတ်မှတ်ခြင်း
let strLength: number = (someValue as string).length;

console.log(strLength); // Output: 17
```

#### 📌 ဥပမာ (၂) - HTML Element Type သတ်မှတ်ခြင်း

```ts
let input = document.getElementById("username") as HTMLInputElement;
input.value = "John";
```

#### 📌 ဥပမာ (၃) – JSON Data ကို Type သတ်မှတ်ခြင်း

```ts
type User = { name: string; age: number };

let data = '{"name": "Alice", "age": 30}';
let user = JSON.parse(data) as User;

console.log(user.name); // Output: Alice
```

### ❓Unknow Type

unknown သည် any ထက် ပိုမိုလုံခြုံသည်။ အသုံးမပြုမီ type ကို စစ်ဖို့လိုတယ်။

#### 📌 ဥပမာ (၁) – Type စစ်ရမယ်

```ts
let value: unknown = "Hello";

if (typeof value === "string") {
  console.log(value.toUpperCase()); // Output: HELLO
}
```

#### 📌 ဥပမာ (၂) – စစ်ခြင်းမရှိသေးလျှင် error ဖြစ်နိုင်

```ts
let val: unknown = 42;

// console.log(val.toFixed()); ❌ Error ဖြစ်မယ်
```

#### ✅ ပြုပြင်နည်း

```ts
if (typeof val === "number") {
  console.log(val.toFixed()); // Output: 42.00
}
```

### 🚫 never Type

never သည် function တစ်ခုမှ ပြန်မလာတဲ့အခါ သို့မဟုတ် မဖြစ်နိုင်တဲ့အခြေအနေတွေအတွက် အသုံးပြုသည်။

#### 📌 ဥပမာ (၁) – အမှားပေးသော function

```ts
function throwError(message: string): never {
  throw new Error(message);
}
```

#### 📌 ဥပမာ (၂) – အဆုံးမရှိ loop

```ts
function infiniteLoop(): never {
  while (true) {}
}
```

#### 📌 ဥပမာ (၃) – Switch ထဲမှာ default ကို never သုံးခြင်း

```ts
type Shape = "circle" | "square";

function getArea(shape: Shape) {
  switch (shape) {
    case "circle":
      return Math.PI * 1 * 1;
    case "square":
      return 1 * 1;
    default:
      const _exhaustiveCheck: never = shape;
      return _exhaustiveCheck;
  }
}
```

### 📝 အနှစ်ချုပ်

| အမျိုးအစား     | ရည်ရွယ်ချက်                               | လုံခြုံမှု  |
| -------------- | ----------------------------------------- | ----------- |
| `any`          | Type Checking ပိတ်ထားခြင်း                | ❌ မလုံခြုံ |
| `unknown`      | အသုံးမပြုခင် Type စစ်ရမယ်                 | ✅ လုံခြုံ  |
| `never`        | ပြန်မလာတဲ့ Function / မဖြစ်နိုင်တဲ့ Logic | ✅ လုံခြုံ  |
| `as` Assertion | မိမိသိတဲ့ type အတိုင်း တင်ပြပေးခြင်း      | ⚠️ သတိထား   |
