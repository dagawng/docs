## ğŸ”¹ Arrays & Objects

### Basic array types

```typescript
let nums: number[] = [1, 2, 3];
let words: Array<string> = ["hello", "world"]; // Generic syntax
let flags: boolean[] = [true, false, true];
```

### array with union types

```typescript
let mixed: (string | number)[] = ["hello", 42, "world", 100];
```

### array with objects

```typescript
let users: { name: string; age: number }[] = [
  { name: "John", age: 30 },
  { name: "Jane", age: 25 },
];
```

### Basic Object Type

```typescript
let person: {
  name: string;
  age: number;
  isStudent: boolean;
} = {
  name: "John",
  age: 28,
  isStudent: false,
};
```

### Optional Properties

Properties that may or may not exist, marked with ?:

```typescript
let person: {
  name: string;
  age?: number; // optional
} = {
  name: "Alice",
};
```

### Readonly Properties

Properties that cannot be changed after initialization:

```typescript
let person: {
  readonly id: number;
  name: string;
} = {
  id: 123,
  name: "Bob",
};

person.id = 456; // âŒ Error: Cannot assign to 'id' because it is readonly.
```
